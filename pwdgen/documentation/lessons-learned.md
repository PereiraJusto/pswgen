# Lições Aprendidas - Gerador de Senhas

## 1. Pontos Positivos
- **Eficiência do Vanilla JS:** O projeto demonstrou que para utilitários focados em performance e segurança, a ausência de frameworks reduz drasticamente o tempo de carregamento e a superfície de ataque.
- **Metodologia Factory:** O uso de etapas bem definidas garantiu que a implementação fosse rápida, pois todas as decisões técnicas e de design já haviam sido validadas nas etapas anteriores.
- **Segurança Nativa:** A utilização da Web Crypto API simplificou a necessidade de bibliotecas externas de criptografia.

## 2. Oportunidades de Melhoria
- **Internacionalização (i18n):** O MVP foi construído apenas em Português; uma versão multi-idioma seria um diferencial.
- **Acessibilidade:** Embora semântico, o projeto pode ser aprimorado com mais atributos ARIA para leitores de tela avançados.

## 3. Conclusão Técnica
O projeto valida o conceito de "Leveza e Segurança" através de padrões web nativos.
